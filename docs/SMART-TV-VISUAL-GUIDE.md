## 📱 Remote Control Navigation

### Using TV Remote:

```
                    [▲]
                     │
              [◄] ── ● ── [►]    ← D-pad for navigation
                     │
                    [▼]

               [  OK/ENTER  ]     ← Confirm selection
               [   BACK     ]     ← Go back
```

**Navigation Order:**

1. Language dropdown
2. Voice gender dropdown
3. Rate slider
4. Pitch slider
5. Volume slider
6. Token input field
7. Counter input field
8. **Announce button** ← Most important!
9. Control buttons (Pause/Resume/Stop)

**Tips:**

- Use arrow keys to move between elements
- Press OK/ENTER to activate buttons
- Press BACK to exit fullscreen

## ❓ Troubleshooting Visual Guide

### Problem: No Audio After Clicking

**What you see:**

```
✓ Audio enabled! Making announcement...
```

**But no sound? Check:**

1. 📺 TV Volume - Is it muted? Turn up volume
2. 🔌 TV Speakers - Are they enabled? Check TV audio settings
3. 🌐 Internet - Is connection stable? Check WiFi icon
4. 🔄 Refresh - Press refresh on browser and try again

---

### Problem: Button Not Pulsing

**What you should see:**

```
╔════════════════════════════════════╗
║  🔊 ENABLE AUDIO & ANNOUNCE  ✨    ║  ← Should be animated
╚════════════════════════════════════╝
```

**But it's static? Try:**

1. Refresh the page (F5 or refresh button)
2. Close and reopen browser
3. Clear browser cache

---

### Problem: Can't Navigate with Remote

**Try this:**

1. Use mouse/keyboard if available
2. Try different browser (Chrome recommended)
3. Check for browser updates in TV settings
4. Use touch screen if TV supports it

## 💡 Pro Tips for Best Experience

### For Clearer Audio:

- Use English language for best voice quality
- Set rate to 0.9 (slightly slower)
- Keep volume at 1.0 (max)
- Enable notification sound for attention

### For Better Display:

- Use fullscreen mode (F11)
- Hide control panel (toggle button) for cleaner view
- Increase browser zoom if text too small
- Keep good lighting on screen

### For Reliable Operation:

- Use wired internet connection
- Keep browser tab active (don't minimize)
- Leave page loaded continuously
- Don't navigate away during announcements
